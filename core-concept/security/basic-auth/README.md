# Basic Auth

Exemples Basic Auth pour la taverne, avec trois variantes d'identite :

- Utilisateurs embarques (in-memory)
- Utilisateurs en base via JDBC (Elytron JDBC)
- Utilisateurs en base via JPA (security-jpa)

### [Basic Auth - Embedded users](basic-auth-tutorial)
Protection de la reserve de nourriture et de biere avec utilisateurs embarques.

### [Basic Auth - JDBC](basic-auth-jdbc-tutorial)
Meme scenario avec utilisateurs stockes en base via JDBC.

### [Basic Auth - JPA](basic-auth-jpa-tutorial)
Meme scenario avec utilisateurs stockes en base via JPA.
